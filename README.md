# Cabinet Dentaire Laroque & Fontenay – Site Démonstrateur

Site vitrine pour cabinet dentaire spécialisé en implantologie et esthétique dentaire à Lyon.

## Aperçu

Démonstration de site web premium dédié aux cabinets dentaires français. Conçu autour d'une approche éditoriale (typographie soignée, asymétrie réelle, palette minérale) pour rompre avec les codes génériques des sites médicaux.

**Note** : ce projet est un démonstrateur de prospection. Le cabinet, les praticiens, l'adresse et le numéro de téléphone sont fictifs.

## Stack technique

- HTML5 sémantique
- CSS3 (variables CSS, Grid, Flexbox, media queries)
- JavaScript vanilla
- GSAP pour les animations du hero
- Polices : Playfair Display (display) + Inter (texte)

## Fonctionnalités

- Bilingue FR / EN (toggle instantané)
- Responsive complet : mobile, tablette portrait, tablette paysage, desktop
- Hero vidéo asymétrique avec animations GSAP
- 11 sections éditoriales (équipe, services, méthode, cas cliniques, technologies, témoignages, FAQ)
- Pages internes en placeholders (démonstrateur, pas site de production)
- Images optimisées WebP, vidéo compressée H.264
